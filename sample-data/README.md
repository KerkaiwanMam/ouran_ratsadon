# Sample SME data

ไฟล์ตัวอย่างสำหรับทดสอบ Business Layer (อัปโหลดที่ `/upload`) — ทั้งหมดอยู่ในรูปแบบ Excel template เดียวกัน:

| คอลัมน์ | คำอธิบาย |
|---|---|
| วันที่ | รูปแบบ `YYYY-MM-DD` |
| รายการ | คำอธิบายรายการ (ใช้สำหรับ auto-categorize) |
| หมวดหมู่ | ไม่บังคับ — เว้นว่างได้ ระบบจะจัดหมวดหมู่ให้อัตโนมัติ |
| จำนวนเงิน | ตัวเลข บวก = รายรับ, ลบ = รายจ่าย |
| ประเภท | `income` หรือ `expense` |

## ไฟล์ตัวอย่าง

- **sme-restaurant.csv** — ร้านอาหาร (รายได้หน้าร้าน + เดลิเวอรี่, ต้นทุนวัตถุดิบ, มีรายการ spike ค่าอุปกรณ์)
- **sme-ecommerce.csv** — ร้านค้าออนไลน์ (หลายช่องทางขาย Shopee/Lazada/เว็บไซต์, ค่าธรรมเนียมแพลตฟอร์ม, แคมเปญโฆษณาตรุษจีนเป็น outlier)
- **sme-consulting.csv** — บริษัทที่ปรึกษา (รายได้ตามโครงการ, ค่าเดินทางดูงานต่างประเทศเป็น outlier)

แต่ละไฟล์มีข้อมูล 2 เดือน ออกแบบให้มีอย่างน้อย 1 รายการที่ควรถูกตรวจจับโดย leak detection (Outlier rule, z-score > 2.5 SD) เพื่อสาธิตฟีเจอร์นี้

> หมายเหตุ: ไฟล์เป็น `.csv` (เปิดได้ใน Excel/Google Sheets โดยตรง) ใช้คอลัมน์ชุดเดียวกับ `sme-template.csv` ใน `apps/web/public/templates/`
